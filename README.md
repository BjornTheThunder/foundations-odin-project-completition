## Description
The Foundations course covered the basics for the start of any web developer journey: covering the basics of HTML, CSS and JavaScript.

**Key concepts learned:**
* Semantic HTML5 & CSS Layouts (Flexbox/Grid)
* JavaScript Fundamentals
* Git/GitHub Workflow
* Problem Solving & Computational Thinking
